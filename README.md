## Technology

**1. Backend**
  - Java 11
  - Spring Boot 2.7
  - Spring MVC Model
  - Spring Data, Hibernate
  - Spring Security

**2. Database**
  - MySQL

**3. Other Technology**
- RESTful API
- JSON Web Tokens
- jQuery Ajax call API
- Gson
- Thymeleaf
- BCryptPasswordEncoder for Authen
- JavaMail
- OAuth 2.0 Google APIs (updating...)
- Paypal integration (updating...)
- Cache (updating...)
